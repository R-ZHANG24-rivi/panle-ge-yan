# 攀了个岩 · 贴图换装规范

> 目的：把当前 IK 骨骼与贴图的比例关系固定下来，方便以后更换发型、衣服、裤子等贴图时保持一致，不用重新调参。
>
> 一句话原则：**骨骼尺寸固定，贴图去贴合骨骼。** 换图时只要保证「新图尺寸 + 图内锚点位置」和旧图一致，比例就自动保持不变。

---

## 一、三层比例体系

比例由三层决定，全部集中在 `game.js` 的 `CONFIG` 里，换图基本不需要动代码。

### 第一层：IK 骨骼尺寸（永远不要改）

IK 用这些长度解算关节点（肩/肘/髋/膝/踝），决定骨架大小：

| 常量 | 值 | 含义 |
|---|---|---|
| `upperArmLength` | 36 | 上臂骨长（肩→肘） |
| `forearmLength` | 36 | 前臂骨长（肘→腕） |
| `thighLength` | 45 | 大腿骨长（髋→膝） |
| `shinLength` | 45 | 小腿骨长（膝→踝） |
| `torsoLength` | 52 | 躯干长（肩线→髋线） |
| `shoulderWidth` | 28 | 肩宽 |
| `hipWidth` | 22 | 髋宽 |
| `headRadius` | 14 | 头部基准半径 |

**换图时这些值保持不变。** 改动它们会改变整个骨架比例，不是换图该做的事。

### 第二层：贴图缩放系数（比例"总开关"，已固定）

这是你要"定下来的比例"。想整体调大/调小某部位，只改这里一个数字，**攀爬 / 下落 / 试衣间三态同步生效**：

| 常量 | 当前值 | 作用 |
|---|---|---|
| `headSpriteScale` | 1.3 | 头部（头发/表情/刘海/配饰）整体缩放 |
| `shirtSpriteScale` | 0.75 | 上衣缩放（实际绘制 = `0.26 × 本值`） |
| `pantsSpriteScale` | 0.75 | 裤子（大腿/小腿）宽度缩放 |
| `beltSpriteScale` | 0.7 | 腰带缩放（实际绘制 = `0.30 × 本值`） |
| `beltOffsetFromShirt` | -7 | 腰带相对上衣底端的纵向偏移（负=上移） |
| `thighKneeOverlap` | 0.95 | 大腿贴图长度微调（只改长度，用于膝盖衔接） |

### 第三层：贴图原始尺寸 + 锚点

这是换图时**唯一需要对齐**的东西。见下面各部位规则。

## 二、换图规则（按部位分类）

不同部位的贴图对齐方式不同，规则严格程度也不同。

### ① 头 / 发型 / 表情 / 配饰（最宽松）

- **绘制方式**：`drawEndpointSprite` — 以头顶端点为中心，等比缩放（含 `headSpriteScale`）。
- **换图要求**：
  1. 新图与旧图**画布尺寸完全相同**（同样的 PNG 宽高）。
  2. **人物内容在画布中的居中位置、大小保持一致**（头顶留白、五官位置不变）。
  3. 用**相同文件名**覆盖到 `assets/player/`，`headSpriteScale` 不用动。

### ② 上衣（宽松）

- **绘制方式**：`drawBodyPartSprite` — 图片几何中心对齐，比例 = `0.26 × shirtSpriteScale`。
- **换图要求**：同尺寸 + 人物在图中居中位置一致，同名覆盖即可。

### ③ 腰带（中等）

- **绘制方式**：`drawBodyPartSpriteAtAnchor` — 以图内像素锚点 `{x:150, y:98}` 对齐，比例 = `0.30 × beltSpriteScale`。
- **换图要求**：
  1. 新 `belt.png` 同尺寸。
  2. **图内腰扣中心仍位于像素坐标 `(150, 98)`**。
  3. 若腰扣宽度变化，连接带起点在 `drawBeltLegConnections` 里的 `buckleSideX`（当前 `17 × beltScale`，腰扣半宽约 17px）需同步微调。

### ④ 大腿 / 小腿 / 上臂 / 前臂（最严格）

- **绘制方式**：`drawAnchoredSegmentSprite` — 把贴图里 `sourceStart → sourceEnd` 这段"骨骼线"拉伸对齐到实际骨骼两点。
- **换图要求**：
  1. 新图同尺寸。
  2. **图内起点锚点（髋/肩）和终点锚点（膝/腕）的像素坐标必须与旧图完全一致**，否则会错位或斜切。
  3. 若造型确实改变了骨骼线位置，就重新测量像素坐标并同步到代码里的锚点（见下方清单），**攀爬态和下落态两组都要改**。

## 四、腰带连接带（代码绘制，非贴图）

`drawBeltLegConnections` 用两根线连接腰带与裤腿：

- **上端**：腰带黄色腰扣左右两侧（`±17 × beltScale`，纵向在腰带带体中心 `beltY − 50.5 × beltScale`）。
- **下端**：裤腿深色线段中点（用大腿贴图源锚点映射，攀爬/下落两态一致）。
- **颜色**：`#1687bf`（随腰带主色）。
- **图层**：连接带在下、腰带贴图在上（腰带盖住连接带上端）。

换图注意：换裤子若深色线段位置变了，需同步下端映射锚点；换腰带若腰扣宽度变了，调 `buckleSideX`。

---

## 五、实操建议

1. **做一套"贴图模板"**：把每个部位当前的 PNG 各留一份作为基准模板，画新图时直接在模板上改造型，**不动画布尺寸和锚点位置**。这是保持比例最省事的办法。
2. **换图后自检**：进游戏看攀爬 / 下落 / 试衣间三态，重点看膝盖/肘部衔接、头部大小、腰带位置。
3. **改锚点务必两处同步**：肢体锚点在攀爬态和下落态各有一份，改一处另一处也要改。
