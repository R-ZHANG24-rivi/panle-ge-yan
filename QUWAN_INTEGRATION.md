## 本地预览保护

当页面通过 `file://`、`localhost` 或 `127.0.0.1` 打开时：

- 大同事件只输出到浏览器 Console，不发送正式数据；
- 不请求正式排行榜，也不会弹腾讯新闻登录；
- 页面会提示“本地预览不会请求正式排行榜”。

正式登录、分享面板、关闭 WebView、右滑退出控制和排行榜写入只能在腾讯新闻 App 的 HTTPS 测试地址中验证。

## 顶部栏策略

- 默认不带 `disabletitlebar=1`：保留腾讯新闻原生标题栏；游戏内“回到开始界面”和分享按钮仍然可见。
- 端内带 `disabletitlebar=1`：开始页增加自定义关闭按钮，并把顶部安全区提高到 60px。
- 腾讯新闻端内分享调用 `setShareInfo/showShareMenu`；微信、QQ、浏览器等端外环境降级到系统分享或复制链接。

## 六套岩点主题

- `theme01` 至 `theme06` 全部保留；
- 首屏只读取当前一套主题，避免六套资源同时阻塞首屏；
- 开始游戏后在后台预载下一套，点击重新开始时按顺序切换，六套循环使用；
- 游戏进行中的左上角返回按钮只返回游戏开始界面，不会关闭网页。

## 正式接口

- 提交成绩：`POST https://api.prize.qq.com/activity/ranking`
- 获取榜单：`POST https://api.prize.qq.com/activity/ranking_board`
- 用户进入后先正常游戏，仅在打开排行榜、提交成绩或领取奖励等身份操作前检查 `getUserInfo().account`；未登录时调用 `login({ type: "all", from: "game_h5" })`。登录成功后获取登录方式、昵称、头像、用户 ID 与过期状态并继续原动作；取消登录则保留游客游戏结果且不提交成绩。登录成功后优先读取 Cookie 中的 `news_token`，拿不到时从 `getUserInfo()` 返回值读取，再自动计算请求字段 `bknSign`。
- 排行榜提交字段为 `activityId`、`cumulativeRanking`、`rankingSize`、`score`、`bknSign`，并通过 JSON 字符串格式的 `extra` 保存本局高度（米）和用时（秒）。榜单会逐条解析所有用户的 `extra`、嵌套 `data/meta` 及兼容的顶层高度/用时字段；后台未返回这些数据时，本人且分数与本机最佳一致的榜单行使用本机记录回填，其他旧记录显示 `--`。测试榜单在首位登录用户成功提交成绩后由后台为该 `activityId` 自动创建。
- 已登录且本机已有历史成绩时，首次打开排行榜会先提交本机历史最佳成绩来初始化榜单，再展示接口返回的排名；`31032` 读取错误会转换为“完成一局后即可参与排行”，提交仍返回 `31032` 时提示联系活动管理员开通后台权限。
- 排行榜请求超时为 8 秒；只读榜单遇到超时、网络错误、`429` 或 `5xx` 时自动重试一次，空昵称账号不进入可见列表。榜单最多返回 50 名，可在 Canvas 面板内上下拖动，底部本人排名固定显示。
- 登录态变为未登录、登录过期或用户 ID 变化时会清空缓存的 `news_token`，避免账号切换后沿用旧签名。

## 大同 SDK

- 大同 SDK 在 `index.html` 中静态引入，平台适配层不再动态创建 SDK 脚本；
- Canvas 操作继续使用 `reportEvent` 手动上报，页面根节点保留 `dt-pgid`；
- 上线前用 Whistle 确认页面、按钮、游戏结果、登录、分享和排行榜事件均只上报一次。

## 小屏适配

- 世界画面继续按 `375 × 812` 等比缩放，不改变玩法坐标；
- 当画布显示比例小于 1 时，Canvas 文字补偿到物理显示不低于 12px；
- 顶部工具栏补偿到约 40px 物理尺寸，所有小控件点击热区同步扩大到至少 40px。

## 上线前必须真机确认

1. iOS/Android 登录、取消登录和登录后自动提交成绩；
2. `setShareInfo/showShareMenu` 是否拉起原生分享面板；
3. 原生标题栏与 `disabletitlebar=1` 两种链接是否符合预期；
4. 大同页面及 Canvas 手动事件是否在 Whistle 中发出；
5. 排行榜后台活动配置、最高分限制和正式数据返回结构；
6. 切后台后音乐暂停，返回页面后按静音偏好恢复。
7. 排行榜后台是否保留并返回自定义 `extra.height`、`extra.duration`；已有旧榜单记录需在新版重新提交成绩后才会补齐高度和用时。
